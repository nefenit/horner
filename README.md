# horner
Evaluates polynomial using Horner's scheme (n additions and n multiplications)

## Formal specification

| Input                                                   | Output     | Time Complexity | Space Complexity |
|:-------------------------------------------------------:|:----------:|:---------------:|:----------------:|
|X: { (x, (a<sub>n</sub>)) &isin; &#8477;<sup>n+1</sup> } | Y: &#8477; | O(n)            | O(1)             |
